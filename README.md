# Python Calculator

## Description

This is a simple Command Line Interface (CLI) Calculator developed using Python. It performs basic arithmetic operations such as Addition, Subtraction, Multiplication, and Division.

## Features

* Addition
* Subtraction
* Multiplication
* Division
* Division by zero handling
* User-friendly command-line interface
* Repeats until the user chooses to exit

## Technologies Used

* Python 3
* Visual Studio Code

## Project Structure

```
Calculator_Project/
│
├── calculator.py
└── README.md
```

## How to Run the Project

1. Open the project folder in VS Code.
2. Open the terminal.
3. Run the following command:

```
python calculator.py
```

4. Follow the on-screen instructions.
